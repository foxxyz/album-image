album-image
===========

Lookup images from the Musicbrainz/Cover Art Archive via CLI.

Requirements
------------

 * Node 18+ (uses native `fetch`)

Usage
-----

### By album name alone

```
npx album-image drukqs
```

### By artist and album name

```
npx album-image Prodigy "the fat of the land" 
```
